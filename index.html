Mengganti Karakter Anime

Untuk mengganti karakter anime:

  1. Ganti URL JSON animasi Lottie di file JavaScript
  2. Cari file JSON animasi di LottieFiles
  3. Ganti parameter di lottie.loadAnimation()

Menggunakan Bitly API Key

Untuk menggunakan Bitly API Key pribadi:

  1. Daftar di Bitly
  2. Dapatkan API Key dari dashboard Bitly
  3. Masukkan API Key saat memilih layanan Bitly
  4. Atau set default di file JavaScript
